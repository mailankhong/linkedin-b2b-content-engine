This file contains real examples of every hook format that works on LinkedIn. Use it to:

## Width Scoring Guide

When estimating mobile readability:

- Narrow letters (i, l, 1, !, ., ,, ;, :, ', j, f, t, r): 1.0
- Space: 1.2
- Regular lowercase (a-z except i, j, l, f, t, r, m, w): 2.2
- Numbers (0, 2-9): 2.2
- Wide lowercase (m, w): 2.8
- Uppercase narrow (I, J, L): 1.5
- Uppercase regular (A-H, K, N-V, X-Z): 2.6
- Uppercase wide (M, W): 4.0
- Common punctuation (-, (, ), ?, $, &, @, "): 1.5
- Emoji: 4.0

Mobile budget: ~110 width units per line before wrapping.
